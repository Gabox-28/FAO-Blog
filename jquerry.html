JQuery

JQuery es una librería de JavaScript que facilita su uso


Para comenzar a utilizar JQuery, primero deberá indicar en el archivo índice de su
HTML que estará implementado esta librería. Para lograr esto, diríjase al final del elemento
"body" y escriba el siguiente código:

<body>

   . . .

<script src= "https://code.jquery.com/jquery-3.6.1.min.js"
   integrity="sha256-o88AwQnZB+VDvE9tvIXrMQaPlFFSUTR+nldQm1LuPXQ="
   crossorigin="anonymous"></script>

</body>

Iniciando JQuery


Una vez implementada la librería de JQuery, procedemos a utilizarla. Para esto,
cree un archivo de JavaScript y escriba el siguiente segmento de código. Dentro de este
segmento de código se utilizará la librería de JQuery

$(document).ready( function (){

//Los métodos de JQuery se usan aquí

});

Acerca de JQuery

La biblioteca jQuery posee las siguientes características:

  • Manipulación de HTML/DOM

  • Manipulación de CSS

  • Métodos de eventos HTML

  • Efectos y animaciones

  • AJAX

  • Utilidades

Detector de eventos de JQuery


JQuery permite definir acciones si algún elemento dentro del HTML llegase a cumplir
con ciertos eventos. Primero, los elementos dentro de HTML deben de estar declarados con id

<body>

   . . .

<button name="boton1" id="boton1" type="button"> Presionar </button>
</body>

Para establecer un detector de eventos, dentro del "Document ready", declaramos el siguiente
código

$(document).ready( function (){

 $("#boton1").click( function () {
   let x = 10;
   console.log(x);
 })

});

Efectos visuales en JQuery


Existen varios efectos visuales en JQuery para la animación de los elementos HTML.
Uno de estos es el "Fade" o la difuminación, Este permitirá que los elementos aparezcan o desaparezcan
con un efecto animado de difuminación

$(document).ready( function (){

// Mostrar con difuminación
 $("#boton1").click( function () {
   $("#div1").fadeIn();
 })

// Ocultar con difuminación
 $("#boton2").click( function () {
   $("#div2").fadeOut();
 })

});

HTML y JQuery


JQuery no solo puede interactuar con los elementos de HTML, también puede alterar el documento
completo con funciones como "remove()" que pueden eliminar elementos HTML

$(document).ready( function (){

 $("#boton1").click( function () {
    $("#div1").remove();  })

});

AJAX en JQuery


AJAX permite hacer alteraciones en el sitio actual sin tener que refrescar la página, por medio
de acceso a un servidor. Por ejemplo, si se desea cambiar un div usando los archivos almacenados en x
sitio se usa el método "load()"

$(document).ready( function (){

 $("#boton1").click( function () {
    $("#div1").load("file.txt");
 })

});
